# SlotStatus

Lightweight durability indicators for the **World of Warcraft Character Frame**.

SlotStatus adds thin durability bars aligned with each equipment slot so players can instantly see the condition of their gear without hovering every item.

## Features
- Per-slot durability bars aligned with equipment slots  
- Automatic updates when gear durability changes  
- Color-coded durability states  
- Lightweight and minimal UI footprint
---

## Durability Colors

| State | Color |
|------|------|
| ≥ 75% | Green |
| ≥ 40% | Yellow |
| < 40% | Red |

---

## Installation

1. Download the addon
2. Place the folder inside:

```
World of Warcraft/_classic_/Interface/AddOns/
```

3. Reload the UI:

```
/reload
```

---

## Compatibility

Designed for **World of Warcraft – The Burning Crusade era UI**.

This era of the game does not include built-in durability visualization, requiring players to inspect each item individually. SlotStatus provides a slot-aligned overview directly in the character window. :contentReference[oaicite:0]{index=0}
