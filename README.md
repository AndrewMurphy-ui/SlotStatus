# SlotStatus

Lightweight durability bars for the World of Warcraft Character Frame.

SlotStatus adds thin durability indicators aligned with each equipment slot so players can instantly see the condition of their gear without hovering each item.

## Features

- Per-slot durability bars aligned with equipment slots
- Automatic updates when gear durability changes
- Color-coded durability states
- Tooltip extension showing item level
- Tooltip extension showing total repair cost
- Lightweight and minimal UI impact

## Durability Colors

- Green ≥ 75%
- Yellow ≥ 40%
- Red < 40%

## Installation

1. Download the addon
2. Place the folder inside:

World of Warcraft/_classic_/Interface/AddOns/

3. Reload the UI:

/reload
